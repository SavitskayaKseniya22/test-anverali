Test assignment

## How to use
1. Clone the repo
``` bash
git clone https://github.com/SavitskayaKseniya22/test-anverali.git
```

2. Change directory
``` bash
cd test-anverali
```

3. Install dependencies
``` bash
npm install
```

4. Start the dev server
``` bash
npm run dev
```
